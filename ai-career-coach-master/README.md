# AI Career Coach 🚀

AI Career Coach is a web application that helps users get career guidance using AI.
The platform analyzes user skills, interests, and goals to provide personalized career suggestions, learning resources, and improvement tips.

---

## 📌 Features

* 👤 User Authentication (Register & Login)
* 🤖 AI-Powered Career Suggestions
* 📊 Skill Analysis
* 📚 Learning Resource Recommendations
* 🎯 Personalized Career Roadmap
* 📱 Responsive UI

---

## 🛠 Tech Stack

**Frontend**

* React (Vite)
* Tailwind CSS

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB

**Other Tools**

* OpenAI / AI API
* Axios

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/ai-career-coach.git
```

### 2️⃣ Navigate to project folder

```bash
cd ai-career-coach
```

### 3️⃣ Install dependencies

Frontend

```bash
cd client
npm install
```

Backend

```bash
cd server
npm install
```

### 4️⃣ Create `.env` file

Example:

```
PORT=5000
DATABASE_URL=your_mongodb_connection
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=
NEXT_PUBLIC_CLERK_SIGN_IN_URL=/sign-in
NEXT_PUBLIC_CLERK_SIGN_UP_URL=/sign-up
NEXT_PUBLIC_CLERK_AFTER_SIGN_IN_URL=/onboarding
NEXT_PUBLIC_CLERK_AFTER_SIGN_UP_URL=/onboarding
GEMINI_API_KEY=
```

### 5️⃣ Run the project

Backend

```bash
npm run dev
```

Frontend

```bash
npm run dev
```



Example:

* Home Page
* Dashboard
* Career Suggestions
